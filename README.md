# QuantPortfolio

QuantPortfolio is a C++ portfolio management project for modeling financial assets such as stocks, bonds, and cash. The project is designed to practice object-oriented C++ through a finance-focused application.

## Features

Current:

- Abstract `Asset` base class
- `AssetType` enum for asset categorization
- Concrete `Stock` asset class
- Stock valuation using shares and price per share
- CMake-based build setup
- Organized `include/` and `src/` structure

Planned:

- `Cash` and `Bond` asset classes
- `Portfolio` class for managing holdings
- Total portfolio value calculation
- Asset allocation summaries
- Command-line interface
